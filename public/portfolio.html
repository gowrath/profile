My Projects

Sharesub is a minimal UI React site that uses several technologies: React Hooks, React-Router (v.5; still uses Switch instead of Routes), localStorage (for CRUD functionality to operate without a database (Refreshing the page will keep data in local storage, unlike cookies which are not persistent.), Userfront Authentication API, and Firebase Hosting.

The Gatsby site uses several Gatsby plugins using lazy-loading images. This speeds up loading the page by loading necessary components first. It uses server-side rendering and is optimized for SEO. The images on my blog are personally taken by me in Emeryville and Seoul. The pages load initially, with the pictures being served secondarily. Some challenges I overcame was the initial setup on Google Cloud with Firebase store. This required getting familiar with the Firebase CLI, setting up authentication, uploading files/assets, understanding buckets and deployment. Dark mode is available at the bottom of the page. I am also an active contributor to the open-source repo.

Seat Master is a classroom management app created by my colleague that I have pushed UI updates to. I work with the metrics component and I styled using Bootstrap. I also troubleshooted the form logic and button behavior. Some interesting use cases might be to monitor student progress on assignments and assign them non-letter grades. As someone with experience in teaching, I feel like this might be a more holistic method of education.
